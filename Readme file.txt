Zip contains following files

Project Presentation Link.docx :- contains video URL link of project presentation
cart.sql - database schema , create "cart" database in mysql and than import schema cart.sql to "cart" database 
Report.docx - Fianl report of project




<<---------------------------------------------Instruction to run project------------------------------------- >>


1. Install XAMPP.
2. Install Eclipse from official website.
3. Import the project from our source code to Eclipse workspace.
4. Open phpMyadmin from XAMPP.
5. Create database called 'cart', and import cart.sql from our source code.
6. Select 'index.jsp' from Eclipse Project Explorer and then right click-run on server.
7. Create tomcat server from menu and click finish.
8. DONE!
